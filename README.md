# Vue + Vuetify Data Analyst Portfolio

This is a Vue.js portfolio website built with Vuetify for showcasing data analyst skills.

## Project Setup

```bash
npm install
npm run dev
```

## Features

- Hero section
- Stats cards for projects, dashboards, datasets
- Projects showcase
- Skills list
- Contact form
- Dark theme

## Technologies

- Vue 3
- Vuetify 3
- Vite
